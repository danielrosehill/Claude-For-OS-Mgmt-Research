Public Claude Code configuration bundle targeted at homelab management: per-service skills,
hooks, slash commands, and agents. Useful as an external reference implementation for the
`skills/` + `agents/` layer as a sysadmin mitigation — i.e. codifying recipes so the model
chooses the right pre-approved invocation instead of improvising a novel shell command that will
trip permission prompts. Contrast point for the workspace's own
workstation-manager / sysadmin-homelab / desktop-manager skill set.
