Community-built `secure-askpass` helper (encrypted-with-SSH-key sudo password + GUI confirm) and
an accompanying discussion about whether Anthropic should bless a pattern for non-interactive
sudo in Claude Code. Closed as not planned. Useful because it shows (a) sudo is a first-class
pain point in automated/Claude-driven sysadmin; (b) Anthropic declines to take a position,
leaving users to roll their own. Relevant when documenting current workarounds and their
fragility.
